Systems of Linear Equations

Audience: Undergraduate linear algebra

  • Goal: Solve $A\mathbf{x}=\mathbf{b}$, classify solution sets, and connect algebra to geometry
  • Outline: elimination, pivots, rank, column space, worked examples

Learning Objectives

  • Write a system as $A\mathbf{x}=\mathbf{b}$ and form $[A\mid\mathbf{b}]$
  • Perform elementary row operations and Gaussian elimination
  • Classify solutions: unique, none, infinite
  • Use rank and column space to test solvability

Definitions & Notation

  • Linear equation: $a_1x_1+\dots+a_nx_n=b$
  • System: $A\mathbf{x}=\mathbf{b}$ with $A\in\mathbb{R}^{m\times n}$
  • Augmented matrix: $[A\mid\mathbf{b}]$

Elementary Row Operations

  • Swap rows: $R_i\leftrightarrow R_j$
  • Multiply a row by nonzero scalar: $cR_i$
  • Add multiple: $R_i\leftarrow R_i+cR_j$
  • All preserve the solution set

Gaussian Elimination (outline)

  • Use row ops to reach Row-Echelon Form (REF)
  • Identify pivot columns and do back-substitution
  • Free variables → parametric solutions

Worked Example 1 — 2×2 (unique)

Solve: $x+2y=5$, $3x-y=4$

  • Augmented matrix: $[\begin{smallmatrix}1&2&\mid&5\\3&-1&\mid&4\end{smallmatrix}]$
  • $R_2\leftarrow R_2-3R_1\Rightarrow [1\ 2\ |\ 5;\ 0\ -7\ |\ -11]$
  • $y=11/7$, $x=13/7$

Worked Example 2 — 3×3 (unique)

System: $x+y+z=2$, $2x+3y+z=5$, $x+2y+2z=4$

  • Augmented matrix and elimination leads to $2z=1$ so $z=1/2$
  • Then $y=3/2$, $x=0$
  • Solution: $(0,\tfrac{3}{2},\tfrac{1}{2})$

Classifying Solutions: Rank

  • If $\mathrm{rank}(A)<\mathrm{rank}([A\mid\mathbf{b}])$ ⇒ no solution
  • If $\mathrm{rank}(A)=\mathrm{rank}([A\mid\mathbf{b}])=n$ ⇒ unique solution
  • If $\mathrm{rank}(A)=\mathrm{rank}([A\mid\mathbf{b}])

Geometric Interpretation

  • $n=2$: lines in $ $ the plane — intersect in point, be parallel, or coincide
  • $n=3$: planes in 3D — intersect in point, line, plane, or empty
  • $\mathbf{b}\in\mathrm{Col}(A)$ iff solvable

In-Class Activity (10 min)

  • Pairs: each pair solves one system (unique / none / infinite)
  • Reduce, classify, and prepare a one-line justification

Homework Exercises

  1. Solve & classify: $x+2y-z=1$, $2x+4y-2z=2$, $-x+y+z=0$
  2. Decide if $A=\begin{pmatrix}1&2\\2&4\end{pmatrix}$, $\mathbf b=(3,7)^T$ is solvable
  3. Solve: $x-y+2z=3$, $2x+y-z=0$, $-x+3y+z=4$

Summary & Tips

  • Row ops preserve solutions; practice the mechanics
  • Rank determines solvability; free variables give parametric families
  • Relate algebra to geometry for intuition
